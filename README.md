# LogsAndMore                
 *v1.0.0 by me*               
**LogsAndMore** creates logs that are nicer and more organized than Python's `logging` module. 
                
## Example                
                
```python                
logger.debug("debug output")                
logger.info("info output")                
logger.success("success output")                
logger.warning("warning output")                
logger.error("error output")                
logger.critical("critical output")                
logger.evento("event output")                
logger.seguridad("security output")                
logger.custom(nivelcus, "custom output")                
```          
## Output                
```text   
[DEBUG]     debug output                
[INFO]      info output                
[SUCCESS]   success output                
[WARNING]   warning output                
[ERROR]     error output                
[CRITICAL]  critical output                
[EVENT]     event output                
[SECURITY]  security output                
[NIVELCUS]  custom output            
```            
## How to install?                
Run:          
```bash              
pip install logsandmore                 
```
